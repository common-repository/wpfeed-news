WordPress Plugin of News Feed from URL for Pages,posts,blog.

== Description ==

A wordpress plugin which provide you powerful functionality to integrate News Feed from various website Feed Url into your site/blog. 
In this plugin you can save Urls from where you need news data and display by using shortcodes given on setting page.

== Installation ==

You can directly upload the plugin from wordpress admin section or use following steps.
1. Upload plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin from the 'Plugins' menu in WordPress
